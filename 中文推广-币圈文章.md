# 别再反复刷 K 线了——我写了个 Telegram 机器人帮我看盘

炒币最累的不是亏钱，是盯盘。

设个心理价位"BTC 再跌 5% 我就抄底"，然后就陷入了两难：盯着屏幕看几个小时等一个不知道什么时候来的机会，还是关掉屏幕然后完美错过。大部分人选择了前者，然后发现一天过去了什么都没干。

市面上也有价格提醒工具，但基本都一个模子：价格到了某个数，发条通知。问题是大多数时候我要的不是"价格到了 X"，而是更条件化的判断——比如"24 小时内波动超过 5%"（说明有了进场信号），或者"两个交易所之间的价差拉大了"（可能意味着插针或流动性事件）。

找不到顺手的，就自己写了一个。

## CoinPing：一个可以在后台替你盯盘的 Telegram Bot

CoinPing 是个 Telegram 上的加密货币提醒机器人，直接搜索 @CoinPingAlertBot 就能用。跟普通价格提醒工具不一样的是，它支持三种提醒条件：

1. **价格阈值** — `BTC 穿过某个价位向上/向下`，最基础的触发条件
2. **24h 涨跌幅** — `BTC 24 小时内波动超过 X%`，用来抓异动不盯盘
3. **跨交易所价差** — `CoinGecko 和 Binance 之间 BTC 价差超过 X%`，抓插针和流动性缺口

设置好条件之后关掉 K 线图，条件触发了 Telegram 会推消息给你。

另外它还有一些日常小功能：`/price BTC` 查实时价格、`/top` 看市值前 10 的币种和 24h 涨跌、`/digest on` 开启每日行情摘要推送。

## 用 Go 写的，零成本跑着

技术栈：Go + telebot v3（Telegram 框架）+ CoinGecko 免费 API + SQLite（纯 Go 实现，不需要 CGo）+ Render 免费层部署。

后台两个 goroutine 跑定时任务：一个每 5 分钟轮询检查所有活跃提醒，一个每分钟检查到点的每日摘要订阅者。单进程、单二进制文件、零外部依赖。

提醒触发后自动禁用（一次性机制，不会反复推送骚扰你）。需要重新开启就再创建一条。

## 免费，目前每人 3 条提醒

现在完全免费，每个用户最多创建 3 条活跃提醒。付费无限提醒版在做，但不会影响现在的免费功能。3 条对大部分个人交易者够用了——一条盯 BTC 大方向，一条抓异动，一条盯价差。

## 我自己的用法

我平时的设置大概是这样：

```
/alert change BTC 5        → 24h 涨跌超过 5% 通知我（抓异动）
/alert price BTC 65000 up  → 突破 65000 叫我（突破确认）
/alert spread BTC 0.5      → 价差超过 0.5% 叫我（可能有插针）
```

设完这三条就可以关掉 TradingView 和交易所 App。Bot 在后台跑着，有信号了自然推送。脑子终于可以空下来做别的事。

## 怎么开始

1. Telegram 搜 @CoinPingAlertBot，点 Start
2. 发 `/alert price BTC 65000 up` 试试第一条提醒
3. `/help` 看完整命令列表
4. 设好条件，关掉 K 线，正常生活

代码在 [github.com/SeigeC/coinping-bot](https://github.com/SeigeC/coinping-bot)，开源可自部署。觉得有用的话给个 Star，有什么想要的提醒条件可以在 issue 里提，我在公开开发。
